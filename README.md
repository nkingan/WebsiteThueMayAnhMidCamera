## 🎯 Giới thiệu
**MIDCAMERA** là website tĩnh phục vụ catalogue, hiển thị đánh giá và hỗ trợ quy trình thuê thiết bị nhiếp ảnh tại Vinh. Khách hàng có thể duyệt danh sách thiết bị, xem giá theo ngày, đặt thuê trực tuyến và đọc đánh giá thực tế từ người dùng đã thuê.

| Thông tin | Chi tiết |
|---|---|
| 📍 Địa chỉ | 45 Bạch Liêu, TP. Vinh, Nghệ An |
| 🏗️ Loại dự án | Static Website (HTML + CSS + JS thuần) |
| 🌐 Ngôn ngữ | Tiếng Việt |
| 📱 Responsive | Desktop · Tablet · Mobile |

## ⚡ Chức năng chính

- 🏠 **Trang chủ** — Hero section, video giới thiệu, testimonials, CTA
- 📷 **Danh sách thiết bị** — Grid sản phẩm, bộ lọc theo loại/hãng/giá
- 🔍 **Chi tiết sản phẩm** — Gallery ảnh, thông số kỹ thuật, giá thuê theo ngày/tuần
- ⭐ **Hệ thống đánh giá** — Xem, lọc theo số sao, có ảnh đính kèm, gửi đánh giá
- 🛒 **Giỏ hàng & Checkout** — Quản lý đơn thuê, thanh toán cơ bản
- 🔄 **Header & Footer động** — Inject từ `js/main.js`, dùng chung toàn site
- 🖼️ **Slider & Gallery** — Thanh trượt đánh giá, lightbox ảnh sản phẩm
